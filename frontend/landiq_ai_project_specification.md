# LandIQ AI - Project Requirements & Structure

## Overview
LandIQ AI is an AI-powered land valuation platform that predicts fair agricultural land prices using Machine Learning and performs ownership and risk assessment.

## Tech Stack
- React.js, Vite, Tailwind CSS
- React Router DOM, Recharts, Framer Motion, Lucide React

## Screen List
1. **Landing Page**: Marketing-focused hero, features, benefits, and stats.
2. **Property Analysis**: 5-step multi-step form capturing Village, Taluka, Soil Type, Infrastructure, Ownership, and Market info.
3. **AI Results Dashboard**: High-impact valuation results, confidence meter, risk assessment cards, and due diligence checklist.
4. **Analytics**: Data visualization suite using Recharts (Valuation trends, Risk distribution, etc.).
5. **Fraud Detection**: Risk badges, document authenticity scores, and anomaly indicators.
6. **Property Location**: Map UI with property markers and nearby facility analysis.
7. **Reports**: Formal documentation of findings with download/print capabilities.

## ML Features (Inputs)
- Village, Taluka, Area_Acre, Soil_Type, Soil_Quality_Score, Land_Type, Irrigated, Road_Touch, Road_Width_Ft, Distance_To_Highway_KM, Water_Source, Number_Of_Owners, Unknown_Registrations, Takeover_Risk, Avg_Price_Per_Acre_Nearby.
